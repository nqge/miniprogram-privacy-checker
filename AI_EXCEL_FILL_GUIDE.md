# AI 增强版 Excel 自动填写工具

## 功能概述

AI 增强版工具使用智能分析算法，从检查结果中提取信息，自动生成更准确、更详细的评估内容。

## 核心特性

### 1. AI 智能分析

#### 多维度分析
- **权限检查分析**：分析权限声明和使用情况
- **API 扫描分析**：分析敏感 API 调用
- **数据流分析**：分析数据流向
- **隐私政策分析**：评估隐私政策完整性

#### 智能评分
- 综合多个检查报告的评分
- 计算总体合规评分
- 识别严重问题和警告

#### 问题提取
- 自动提取关键问题
- 按严重程度分类
- 生成整改建议

### 2. 自评估表智能填写

#### 规则引擎
基于不同的评估类别，使用不同的评估规则：

**隐私政策相关**
- 检查是否存在隐私政策文件
- 评估政策内容完整性
- 检查政策可访问性

**权限声明相关**
- 统计已声明权限数量
- 评估权限使用合规性
- 检查最小必要原则

**数据收集相关**
- 评估数据收集透明度
- 检查用户授权机制
- 评估数据使用规范

**安全保护相关**
- 评估安全措施
- 检查数据保护机制

#### 智能评估说明生成
根据检查结果自动生成详细的评估说明，包括：
- 当前状态描述
- 存在的问题
- 改进建议

### 3. 权限确认单智能填写

#### API 调用分析
- 从 API 扫描结果中提取实际使用的权限
- 识别动态权限调用
- 统计权限使用次数

#### 业务功能推断
基于代码上下文推断权限的业务功能：
- 从文件路径推断（如 `pages/avatar/avatar.js` → 用户头像设置）
- 从函数名推断（如 `chooseLocation` → 位置服务）
- 从代码上下文推断

#### 必要性说明生成
为每个权限生成详细的必要性说明：
- 说明权限的使用场景
- 解释为何需要该权限
- 说明用户触发机制

## 输入要求

工具需要以下检查报告文件（JSON 格式）：

1. **permission_check.json** - 权限检查报告
2. **api_scan.json** - API 扫描报告
3. **dataflow_analysis.json** - 数据流分析报告（可选）
4. **privacy_policy_check.json** - 隐私政策检查报告（可选）

## AI 规则引擎

### 评估规则

```python
# 规则 1: 隐私政策相关
if '隐私政策' in text:
    if has_privacy_policy:
        result = "是"
        description = "已发现隐私政策文件，内容完整，符合要求"
    else:
        result = "否"
        description = "未找到隐私政策文件，建议立即创建并公开隐私政策"

# 规则 2: 权限声明相关
elif '权限' in text or '申请' in text:
    if permission_count > 0:
        result = "是"
        description = f"已声明 {permission_count} 项权限，权限使用符合最小必要原则"
    else:
        result = "否"
        description = "未声明必要的权限，建议在 app.json 中补充权限声明"
```

## 技术架构

### 数据流程

```
检查结果 (JSON)
    ↓
AI 分析引擎
    ├─ 加载多个报告
    ├─ 计算总体评分
    ├─ 提取关键问题
    └─ 生成评估说明
    ↓
Excel 填写
    ├─ 自评估表填写
    └─ 权限确认单填写
    ↓
保存 Excel 文件
```

### 核心算法

1. **评分算法**：加权平均各检查报告的评分
2. **问题提取**：按严重程度分类和排序
3. **规则匹配**：基于关键词匹配评估规则
4. **上下文分析**：从代码路径推断业务功能

## 故障排除

### 问题：AI 评估结果不准确

**原因**：检查报告数据不完整

**解决**：
- 确保运行完整的检查流程
- 检查 JSON 报告文件是否存在
- 验证报告文件格式正确

### 问题：部分权限未填写

**原因**：API 扫描未检测到相关调用

**解决**：
- 检查 api_scan.json 内容
- 确认 API 扫描正常运行
- 手动补充未填写的权限

### 问题：评估说明过于简单

**原因**：规则引擎未匹配到具体规则

**解决**：
- 检查评估点的关键词
- 添加自定义规则
- 手动完善评估说明
